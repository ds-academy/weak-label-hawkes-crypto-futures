## Supplementary Material Boundary

Recommended first submission package:

- Main arXiv source: `main.tex`, `references.bib`, and `main.bbl`.
- Rendered tables: embedded in `main.tex`.
- Rendered figures: tracked under `figures/*.pdf` and referenced by `main.tex`.
- Protocol documents (`K1_*`, `J1_*`, `K2A_*`): do not include in the main
  source package unless a supplementary-material decision is made explicitly.
- Source code and generated artifact repository: link through repository URL
  and archival identifier once those are finalized.

## K3 Boundary

Do not use K3 to strengthen the claim, add new empirical evidence, or modify
the diagnostic Path A conclusion. Any content change should start a separate
manuscript commit before regenerating the arXiv package.

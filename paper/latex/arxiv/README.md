# arXiv Package

This directory contains the generated Path A arXiv package.

Source-of-truth policy:

- `../manuscript.tex` is generated from `../../manuscript.md`.
- `main.tex` is copied from `../manuscript.tex` by `../build_arxiv_package.py`.
- `references.bib` is copied from `../references.bib` by the same builder.
- `main.bbl` is generated by the compile step and tracked so the upload source
  includes the materialized bibliography.
- `figures/*.pdf` are generated by the same builder from the current J0/J1
  artifact data.
- Do not hand-edit `main.tex`, `references.bib`, or `main.bbl`; regenerate
  them from the Markdown and BibTeX packaging sources.

Build command:

```bash
python ../build_arxiv_package.py --compile-pdf
```

Packaging boundary:

- K3 is a format/package step, not a content rewrite step.
- arXiv category selection is recorded downstream during submission metadata
  preparation. The current checklist recommends `q-fin.TR` primary with
  `q-fin.ST` cross-list.
- The generated `main.pdf` and disposable LaTeX auxiliary files are
  intentionally ignored. The figure PDFs and `main.bbl` are tracked because the
  arXiv source package needs them.
- See `SUBMISSION_CHECKLIST.md` for the current metadata recommendation,
  compile status, and remaining upload blockers.
